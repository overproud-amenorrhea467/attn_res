## ✨ What this app does

attn_res is a single-file PyTorch app for trying Attention Residuals on your own machine. It brings together:

- Attention Residuals
- Grouped Query Attention
- SwiGLU feed-forward layers
- Rotary Position Embeddings

It is meant for people who want a local Windows app that works with modern transformer ideas without a long setup.

## 🔍 Terms in plain English

- **Attention**: a way for a model to focus on the most useful parts of input
- **Residuals**: extra paths that help information flow through the model
- **GQA**: a memory-saving way to handle attention heads
- **SwiGLU**: a feed-forward layer style used in newer models
- **RoPE**: a way to add position info so the model knows order
